# QuickCrop - 极速批量裁切

一个简单高效的批量图片裁切工具，基于 React 构建，支持多种分辨率预设。

## 功能特点

- **批量处理**：一次上传多个图片文件
- **多种分辨率**：支持 1024x1024、896x1152、768x1344 等常见比例
- **所见即所得**：预览界面实时显示裁切效果
- **自由调整**：拖拽移动位置，滚轮缩放大小
- **快捷键支持**：按 Enter 键快速保存当前图片
- **自动适配**：图片自动按最短边撑满目标区域

## 使用方法

1. 点击「选择文件夹」上传图片
2. 选择目标分辨率
3. 拖拽调整图片位置，滚轮缩放
4. 按 Enter 或点击「下一张」保存
5. 图片会自动下载为 `crop_1.jpg`、`crop_2.jpg` ...

## 在线使用

直接在浏览器中打开 `index.html` 文件即可使用（无需服务器）。

## 技术栈

- React 18
- Tailwind CSS
- Babel (JSX)
